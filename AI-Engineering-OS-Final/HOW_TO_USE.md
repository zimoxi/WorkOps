# HOW_TO_USE.md

## For New Projects

1. Copy all files from this template into the new project.
2. Ask the AI assistant:

```text
Please follow AGENTS.md and initialize this project according to the AI Engineering OS standard.
Create or update README.md, ARCHITECTURE.md, PROJECT_STATE.md, TASKS.md, CHANGELOG.md, DECISIONS.md, and WORKLOG.md based on the current project.
From now on, do not rely on chat history. Keep project files as the source of truth.
```

## For Existing Projects

1. Copy these files into the existing project.
2. Ask the AI assistant:

```text
Please read AGENTS.md and PLAYBOOK.md, inspect the existing codebase, then generate accurate PROJECT_STATE.md, TASKS.md, ARCHITECTURE.md, DECISIONS.md, and WORKLOG.md.
After that, continue development only using project files as the source of truth.
```

## Daily Usage

At the beginning of work:

```text
Read AGENTS.md, PROJECT_STATE.md, TASKS.md, and continue the next task.
```

At the end of work:

```text
Update PROJECT_STATE.md, TASKS.md, CHANGELOG.md, and WORKLOG.md before finishing.
```
